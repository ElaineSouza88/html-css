Marcadores de texto

Uma forma de deixar um trecho do texto marcado, é utilizar a tag MARK.

Alterar a cor dos Marcadores

Essa é uma das funções do CSS, pois se trata de uma estilização.

Para alterar a cor do marcador, basta usar a tag da CSS STYLE, dentro da tag MARK. Dessa forma tud o que estiver dentro desse parâmetro, ficará diferente.

Para que essa parte do texto herde a mesma cor escolhida precisamos criar no HEAD do HTML, um seletor para a marcação MARK, que herdará uma cor padrão para TODAS as demais tags de marcação.
