Usando o target

Esse é o atributo responsável por controlar onde o link externo será aberto

Ele suporta os seguintes valores:

  • _parent -> Similar ao top em uma referencia de janela mãe.
  • nome-do-frame, caso esteja usando frames, indicar o nome da janela que deseja abrir.

Exemplo

  • _blank -> Irá abrir o link em uma nova janela em branco.
