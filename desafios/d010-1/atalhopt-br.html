Atalho para Linguagem

Criado variável emmet

Essa configuração permite já deixar parametrizar a Linguagem PT-BR, sempre que iniciarmos uma estrutura nova em HTML

Procedimento

  1. No VC Code acesse Arquivo
  2. Preferencias
  3. Configurações
  4. Pesquise por: Variáveis emmet
  5. Clique em Adicionar item, e crie a variável LANG e no valor coloque pt-br
  6. Adicinar

    7. Agora toda vez que iniciar a sintaxe com !, já virá parametrizado.
