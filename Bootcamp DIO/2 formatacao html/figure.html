Tags para imagens

Figure

Essa tag é um container que possibilita delimitar um espaço para uma imagem. É uma tag explicativa dos elementos img podendo acrescentar mais de 1 imagem na mesma tag ou agrupar em várias tags auxiliares para esse recurso, veja o exemplo da estrutura abaixo, utilizanodo o DevTools do Google Chrome.

Mesmo que possamos utilizar apenas a tag de img e a tag p, para realizar essa mesma configuração acima, porém não é semântico, utilizar dessa forma.

Tag Picture

Essa tag possibilita múltiplas fontes para a mesma imagem, por exemplo, o atributo media, que possibilita alterar o tamanho da imagem quando a tela for menor que um determinado tamanho.
