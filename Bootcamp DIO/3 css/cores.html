Formas de aplicar cores com CSS

  • Pré definidas = Escrevemos o nome da cor dentro da tag style: color: blue;
  • Palavra Chave (current-color) = atributo usado para respeitar a cor de fundo que foi definida pelo navegador.
  • Hexadecimal = Leva em consideração as propriedades hexadecimais de cada cor utilizando o # + os valores de cada cor. Ex.: #ff0000 representa a cor vermelha.
  • Hexadecimal transparente = É a cor hexadecimal com transparência.
  • RGB = Definição da combinação de cores de Red, Green e Blue, que geram uma quarta cor.
  • RGBA = Acrescenta as combinações de cores RGB o "canal alfa" que representa a transparência dessa nova cor.
  • HLS = Matiz - Representa o grau da cor, Saturação - representa o percentual da saturação de uma cor, e Luminosidade que representa também em valores percentuais, a luminosidade das cores.
  • HSLA = Representa os mesmos elementos do HLS e acrescenta a transparência.
